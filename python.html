Программа обучения

Основной блок

Python Basic

Изучите основы работы с Python, операторы и выражения, циклы, условные операторы, числа, функции, установку и настройку IDE, базовые коллекции, работа с файлами, исключения, основные принципы ООП, итераторы и генераторы, декораторы, библиотеки для работы с данными, основы работы с сетью. Напишите дипломный проект: Telegram-бот для поиска отелей.

Язык запросов SQL

Познакометесь со структурой БД, изучите SQL, научитесь выбору и фильтрации данных, агрегации данных, группировке, относить и соединять таблицы, модифицировать данные, подзапросам, объявлять и изменять структуру данных.

Django

Познакомьтесь с популярным веб-фреймворком Django и с помощью его разработайте свой первый интернет-магазин.

Python Advanced

Изучите Flask, основы работы в Linux, Logger Dogger, многозадачность, базы данных, MVC-фреймворк, CGI server, основы дебаггинга и профилирования. Напишите дипломный проект: Разрабокта задачника по Python.

Дополнительный блок

Система контроля версий GIT

Универсальные знания программиста

Дадим советы: как стать первоклассным программистом, как искать заказы на разработку, личный бренд разработчика, вёрстка email-рассылок, the state of soft skills, как общаться по email и эффективно работать с почтой, повышение своей эффективности, спор о первом языке программирования, Саморазвитие: как я не усидел на двух стульях и нашел третий, data-driven-подход к продуктивности — инсайты из данных миллиона людей.
