Программа обучения

Основной блок

Веб-вёрстка. Базовый уровень

Изучите базовый HTML, CSS, работу с макетом, HTML-разметку, Flexbox, стилизация, продвинутый HTML, CSS, сетки, адаптивность, работу с хостингом, JS для верстальщика.

Веб-вёрстка. Продвинутый уровень

Изучите SVG, анимацию на JS, сборщики, препроцессоры и постпроцессоры, CSS Grid, CMS.

JavaScript. Базовый уровень

Познакомитесь с переменными, работой с числами, строками, условными операторами, массивами, циклами, функциями, объектами, введением в DOM, константами, преобразованием типов, обработкой событий.

PHP

Изучите структуру управления данными, базовые концепции PHP 7, протокол HTTP/HTTPS, пользовательские функции, файлы и каталоги, Cookie/Session, базу данных MySQL, объекты, функции и стандарты по работе с классами, структурой проекта Laravel.

Node.js

Первый "Hello world!", решите базовые задачи, познакоммитесь с базой данных, изучите Real-time & WebSockets, Pro tips.

Как проходит обучение?

Групповые интерактивные лекции

Лекции будут проходить в зуме, где у каждого участника есть возможность задать вопрос и сразу же получить на него ответ. Лекции будут проходить 2 раза в неделю по 1.5 - 2 часа. Каждая лекция включает в себя практические задания и тесты, которые вам нужно будет сделать самостоятельно.

Продолжительность и объем

Курс расчитан на 12 месяцев. Помимо заданий на лекциях вам будет выдаваться дополнительно домашние задания, а также мини-проекты раз в месяц. Каждое ваше действие будет проверено ментором с обратной связью.

По итогу

Вы познакомитесь с веб-разработкой и получите опыт практической работы. У вас будет выполнен проект, который будет проверен ментором. Это хороший плацдарм, который поможет вам в будущем.
